== Description ==

Plugin for displaying a timeline based on a custom field.
The css is customizable from the admin area of the plugin.

Based on Mauro Rocco's "History timeline" v0.3


== Installation ==

If you have never use this plugin or a previous version, you have extract the archive file (.zip) and upload the folder history-timeline in wp-content/plugins of your web space. Than activate the plugin in plugin menu of admin control panel.

Then put **[history_timeline]** where you want to view the timeline.
